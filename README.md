# Mon Portefolio - Léo Bourin

Lien : https://blogleob.fr/

## Pré-Requis

- NodeJs
- ReactJs

## Installation

### Clonnez le répo :

```
$ git clone lien
```

### Placez vous dans le bon dossier

```
$ cd portefolio
```

### Installez les dépendences 

```
$ npm install
```

### Démarrer le projet

```
$ npm start
```

**Une fois le projet démarré, une page web s'ouvre...**
